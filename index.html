Como usar

1. Para testar os projetos no seu computador:

  • Baixe ou clone este repositório.
  • Abra a pasta no seu computador.
  • Clique duas vezes em index.html para abrir o site.
  • Depois clique nos links dos projetos para abrir cada um.

2. Para testar os projetos pelo GitHub:

  • Basta entrar no repositório e abrir os arquivos normalmente.
  • Ou usar o GitHub Pages, caso esteja ativado.

3. Cada link abre um projeto separado feito durante nossos estudos.
